# SV_Models

This repository contains the official implementation, supplementary materials, and appendices for the paper:

> **Stochastic Volatility Models with Long Memory: A Comprehensive Review of Theory, Inference and Applications** > *Qiang Zheng, Shelton Peiris, Jennifer S.K. Chan, Zhenglyu Huang*

---

## 📌 Overview

This repository is organized to help researchers and practitioners replicate our findings, explore the theoretical derivations, and apply long-memory Stochastic Volatility (SV) models to their own financial time-series data.
